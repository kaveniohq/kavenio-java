# Kavenio Java SDK

Official Java SDK for Kavenio social posting and scheduling.

## Installation

```bash
com.kavenio:kavenio-java
```

## Get an API key

Create a Kavenio account from the [dashboard](https://app.kavenio.com), then create an API key from [API Keys](https://app.kavenio.com/api-keys).

```bash
export KAVENIO_API_KEY="<your-api-key>"
```

## Generated SDK

The generated SDK source lives in [generated/java](generated/java).

This SDK currently supports Kavenio social posting and scheduling workflows:

- profiles
- connected accounts
- connect/OAuth flows
- media upload and validation
- posts
- queue scheduling
- post validation

For the complete REST API reference, use the Kavenio API docs.
